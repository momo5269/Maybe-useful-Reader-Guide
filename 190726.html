PDF

 

阅读

　　考虑到各种类型大小的打开/浏览速度、文本复制、书签批注，基本编辑等功能。

　　多数阅读器都能设置背景色（多在访问选项）

 

Acrobat

　　功能最全，对应的启动速度较慢。仅阅读器Acrobat Reader DC（下载记得取消勾选！历史版本）

　Acrobat Pro　目前分为XI和DC两个版本，个人选择的是XI。（可在weibo寻找直装版 (2018SP)）

　MiniPDF　辅助软件，可对同目录下PDF快速切换预览。适合散页偏多。

 

福昕阅读器

　　知名不用多谈。基本功能表现优异，对于（嵌入）字体缺失、结构受损、文档描述错误等情况兼容性极佳，其他阅读器如Acrobat可能会乱码报错无法打开。文本查看器方便查看双层PDF，各版本也有绿化/便携版。

 

　最新稳定版　领鲜版　版本历史

1-3　十分精巧，仅1-3M。各类PE和以前Ghost系统常带，兼容一般。适合PE环境或偶尔使用。

4.3　小巧轻快，10M左右。小概率会有兼容问题，如较新的图文混排。若提示更新，可添加CJK字符支持1/2解决。*

6-7　100M左右，共同BUG：LaTeX编写的较旧PDF可能报错或闪退。另外，从7开始仅有Ribbon界面。

福昕风腾PDF套件金山版 ⭐响应较快，表现稳定，100M左右。有上述6-7的BUG，加载特大文件时操作偶尔崩溃。

xxxxxxxxxx
　打开特大文件时一般不会操作，其余时候崩溃极少，仅添加打印略慢。所以小小推荐[福昕风腾金山版]作为默认。

 

PDF-XChange Viewer

　　阅读、标注等可与福昕匹敌，书签甚至优于福昕风腾金山版。不含OCR仅15M，有原生便携版。但双层PDF的搜索效率较差（要多等会）、只显示绝对页码、访问选项仅可修改纯黑白PDF的背景色。另外，添加打印任务较快。

　　文电通 和PDFXCview类似。各方面优异，输在搜索上（效率差+界面简陋）

 

SumatraPDF

　　麻雀虽小，五脏俱全。小巧轻快（仅6M）原生便携。支持DJVU与EPUB，但epub排版不好，虽有中文显示优化版但仍不算好用。

　　百度阅读器精简版 同样小巧轻快。除极少数双层PDF的加载外（可通过acrobat另存解决）近乎完美，但缺少视图布局（如单页、单页连续、对开的切换选择）不如SumatraPDF好用。

 

UnicornViewer

　　老马开发的适用于PDG/PDF的阅读器，支持图像增强，臻于完美。

　　由于PDF阅读与unicode仅高版本支持，目前难以取得（需论坛用户名），所以放在末尾介绍。

　　最后的公开版本（0.12j）可在国学数典或蓝奏云下载。

书签

　　任意OCR软件连同

　　PdgCntEditor　编辑制作书签，导入导出（下称PCE）

　　PDF-XChange Viewer 　复制粘贴保存书签、制作详细定位书签（此类需PDF补丁丁导出xml）

 

　　原有书签，PDF-XChange Viewer复制粘贴即可，PCE/FreePic2Pdf导出导入亦可。

　　自制书签，分为手工录入定位和OCR目录编辑。

　　

  • 手工　

    • 在当前页直接新建书签、编辑文本即可。建议用PDF-XChange Viewer
    • 页面定位的阅读布局为单页，详细定位阅读布局为连续（实际位置为可视部分顶部）
    • 有些可在豆瓣、购物网站拿到目录，部分超星有则可工具获取，这些用PCE编辑导入

     

  • OCR目录编辑

    • OCR，ABBYY PDFT+稍等直接复制…方法很多请自行选择 。
    • 编辑，PCE拥有所需全部功能，且可保存正则调用。建议直接拖入文件，用其编辑保存书签。
    • 正则，比如[.+?]+替换空白来删除标题页码间的圆点。（或直接替换为Tab空格）
    • PDF补丁丁有OCR目录页转为书签的特色功能。

 

  • 正则表达式

    • 语法常识，可参阅正则表达式必知必会、PCE使用说明

    • 简单介绍删除空格后划分章节可用的正则，“替换为”的空格加号为示意：

      • 右移生成子母关系，根据章节情况决定Tab空格数量，用点/顿号的请举一反三

        正则：第([一二三四五六七八九十百零〇]+|[0-9]+)章　 替换为：Tab空格+第$1章

      • 章节文字后加入空格（我喜欢用全角空格，搜索高亮容易区分）

        正则：第([一二三四五六七八九十百零〇]+|[0-9]+)([编章节])　替换为：第$1$2+空格

    • 若需替换后检查，可用EmEditor等支持搜索全部+结果高亮的编辑器。正则检查PCE自带。

    • 目录文字点击跳转，批量暂无方案。逐个，常见编辑器均支持（可用PDF-XChange Viewer）

    • 之前习惯EmEditor，并未发现PdgCntEditor的正则保存，实是疏忽，万分抱歉。

　　用BeCyPDFMetaEdit打开PDF，完善书名作者标签，创建者 / 制作程序写入自己。

　　一般不需要标记，我改meta是因为处理后等于新版本，收尾加不碍事。同时方便用filemeta的同学检索。这种事情，若非花费大量时间的确不需要标注防偷…真有。用老马的软件生成双层PDF时一般制作程序不作变动。

PDF去水印

　　因为水印的情况较多，无插图、表达差、说明复杂，请见谅。简单操作可以直接看PitStop。

 

基本知识

常见水印

  1. 文字型　页眉页脚有软件名、网址、QQ号等文字（软件如pdf打印机，有时是中央45度（如爱学术））
  2. 图片型　页面中央 / 边角有Logo图像
  3. 写入图片　如右下角灰色暗块PDG字样
  4. 复杂对象　PDF-XChange Viewer虚拟打印为代表

　分辨类型可用老版本福昕编辑器（1.2/2.1/5.0）点选对象

PDF类型

  1. 扫描版即纯图片构成的PDF，通常一页一图。
  2. 文字版即文字构成的PDF，图文混排也属于此类。

 

常用方法

　　导出图片、PitStop、pdftoy流过滤、ComicEnhancerPro（下文略为CEP）PDF水印清理专家等

　　去水印软件通常收费、功能有限或无破解，一般推荐使用Acrobat+PitStop

 

PitStop 可解决扫描版的1 2 4， 文字版的1 2

  • 可选矩形、绘制多边形区域，清除对象类型、是否包含重叠
  • 类 2 若是文字版下面盖图，区域选定后移除对象只选图像即可。
  • 类 4 若极为复杂，可能需要勾选重叠并多次操作。重叠文字可能消失，因此对文字版效果不一定好。
xxxxxxxxxx
选择区域 → 全局更改 → 标准 → 移除 → 移除图形 → 吸管图标选择从选区获取区域 → 运行

 

导出图片 可解决扫描版的1 2 3

  • 类 1 导出图片即可
  • 类 2 导出后若水印有大量相同图片，可用文件查重批量删除
  • 类 3 导出后用CEP批量提高图片亮度，再适当提高伽玛修补表现
  • 类 3 若是彩色CEP 通道→替换能指定阀值替换。也可用通道→通道互换、通道混合先处理
  • 导出原图发现每页多图拼成的，可以用PDF补丁丁的尝试合并相同页面的图片（因此有一定局限）

对开拍照书

基本流程：去边/对切→去灰度阴影→纠斜→去边→居中对齐

  1. 对切前，边缘有较大黑边/色块、图片均大致相同容易去除，可批量裁剪去边。反之则对切后去边

    若比较规则，建议用CEP的手动选区范围

  2. 对切大致相同用分割软件；左右不同，可尝试用晨晨双页扫描图片分割器

  3. 去灰度阴影用CEP调整色阶、亮度/伽马或曲线使图片接近纯黑白

  4. CEP纠斜：若图片情况较好，不需再次去边可直接同时设置开本和居中对齐。

    若不好，则编辑后再居中对齐。（软件都不太好，仅Hornil StylePix 1.14.5 似乎可用）

  5. 以上均可通过PhotoShop录制动作实现，取决于习惯…不过一般人是不会想处理这种书的

  6. PDF可用Acrobat+Quite Imposing裁剪后导出，或省事不再处理直接看。

测试说明

 

一、较旧PDF

　　如《lshort中文版4.2》闪退，但5.0-6.0.1版本均无问题，目前推测与部分CTeX输出文档存在兼容问题。

　　由于CTeX套装已停止维护，其输出的版本现存极少、难以找到合适的档案打开/输出测试，且本人XeTeX 3.14159265-2.6-0.99999(TeX Live 2018/W32TeX) 输出的PDF没有任何问题……因此不再深入兼容测试。

 

二、安卓PDF阅读器

　　日常Adoobe足够，兼容性好。但功能简陋，如不支持裁剪、底色，不太适合大量阅读。

　　只要是支持PDF的阅读器，表现都不错…但也有区别，如下：

  1. 特大文件　300M以上　可拿扫描版辞典与书格古籍试验。

  2. 加载模式　多指图片PDF如何完全显示。有时也包含复杂的文字版、能否模拟翻页。

    • 瀑布式逐行：Adobe Reader
    • 模糊变清晰：XoDo、WPS PDF
    • 空白/进度条：多看、智器 、SideBooks

  3. 翻页跳转　快速翻动、拖动进度跳转时候加载所需时间

  4. 编码兼容　多种中文文字版PDF能否正常打开，仅少数软件支持不行

  5. 附属功能　标注笔记、文件底色、转换合并、朗读等，其中底色修改能够对白色替换的不多。

　　可见强调个人需求、习惯，因此不做过多介绍，请参考酷安应用集的文末、回复区。

 

三、安卓CHM阅读器

　　由于CHM文件年代久远、制作不规范等问题，很难做到较好的兼容性。

？：掌阅我无法翻页，能正确显示第一页标记为此。（除9外估计没问题，也就是成功7个）
